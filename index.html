identifikasi byClassName

By Class Name digunakan untuk mengembalikan semua elemen dengan nilai properti class yang diberikandalam dokumen menentukan nama kelas koleksi.

Kelebihan By Class Name :

1. Dapat memanggil seluruh elemen DOM menggunakan nilai properti class yang diberikan dalam dokumen menentukan nama kelas koleksi.

2. Dapat mengisikan objek seperti array (lebih dari 1 data).

3. Dapat mengembalikan objek seperti array (lebih dari 1 objek).

Kekurangan :

1. Cenderung kurang unik dan harus teliti dalam penamaannya

identifikasi byID

Metode byID menggunakan id untuk mencari elemennya. Jika ada elemen yang memakai id yang sama, elemen yang dituju adalah elemen yang pertama. Digunakan saat programmer ingin mengambil elemen dengan id tertentu (karena id pada elemen tidak boleh sama).

Kelebihan By Id :

1. Dapat menyimpan hasil eksekusi fungsi ke dalam variabel.

2. Dapat mengakses elemen DOM berdasarkan nilai properti id pada elemen.

3. Bersifat unik, sehingga setiap dokumen berbeda id.

Kekurangan :

1. Hanya akan mengembalikan satu elemen DOM.

2. Jika terdapat lebih dari satu elemen dengan id sama, maka elemen yang pertama ditemukan akan dikembalikan.

identifikasi byTagName

Metode getElementsByTagName() ini mengembalikan kumpulan elemen dengan nama tag yang sudah ditentukan, sebagai objek NodeList. Kumpulan elemen ini bisa diakses dengan nomor indeks. Indeks tersebut dimulai dari 0. Digunakan jika programmer ingin melakukan perubahan pada elemen dengan tag tertentu.

Kelebihan Tag Name :

1. Memilih elemen-elemen dengan tag HTML tertentu.

2. Memberikan nilai berupa HTMLCollection.

Kekurangan :

1. Harus paham tentang tagname karna dapat memberikan dampak tertentu
